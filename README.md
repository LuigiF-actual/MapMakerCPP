## Requirements 
### CXX 23 or higher 
### Cmake 3.30.0+
### compiled and tested on Windows with MSVC 19.50+, MinGW-W64 15.2.0 and on Linux mint with GCC 15.2.0

## How to build
### git clone https://github.com/LuigiF-actual/MapMakerCPP.git
### cmake -B build -G Ninja
### Cd MapMakerCPP
### cmake --build build
